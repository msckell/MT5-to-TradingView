# Versión en español

Versión en español del proyecto — **mismas funcionalidades y correcciones que la
versión inglesa** de la raíz (fix de stop loss / línea violeta, regla del take profit,
lectura robusta del log del EA y menú reordenado), con los comentarios y el menú en
español. Usá la que prefieras: las dos se comportan igual.

> Nota: esta versión es de consola. La interfaz gráfica (GUI) está solo en la versión
> inglesa de la raíz.

## Uso

1. Copiá `config.example.json` a `config.json` y editá tus rutas/símbolos.
2. Instalá dependencias: `pip install MetaTrader5 pytz pyperclip`.
3. Ejecutá `python mt5_to_tradingview.py` (o el `.bat`), elegí la semana/día,
   y pegá el prompt en Claude Code con Ctrl+V.

Los créditos del lado de TradingView son del MCP
[`tradingview-mcp`](https://github.com/tradesdontlie/tradingview-mcp) de
[tradesdontlie](https://github.com/tradesdontlie), igual que en la versión inglesa.
